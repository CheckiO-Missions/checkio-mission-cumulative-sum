In this mission you are given a list of numbers. You are required to return a list of the same size, but each value there should be the accumulated sum of the initial list. So, each value of the resulted iterable must be generated by summing the elements from the previous list, starting from current index till the end.

Input: A list of ints.

Output: A list of ints.

Example:

{% if interpreter.slug == "js-node" %} 
accumulateValues([1, 2, 3]) == [1, 3, 6]
accumulateValues([1, 1, 1, 1, 1]) == [1, 2, 3, 4, 5]
accumulateValues([1, 0, 0]) == [1, 1, 1]
accumulateValues([]) == []
{% else %} 
accumulate_values([1, 2, 3]) == [1, 3, 6]
accumulate_values([1, 1, 1, 1, 1]) == [1, 2, 3, 4, 5]
accumulate_values([1, 0, 0]) == [1, 1, 1]
accumulate_values([]) == []
{% endif %}
