You need to generate iterable with accumulated values. Which means that result iterable should have the save same size as a given one, but every element of the result one should be a sum of all values in the initial one, starting with index of the current solution.

Input: A list of ints.

Output: A list of ints.

Example:

{% if interpreter.slug == "js-node" %} 
accumulateValues([1, 2, 3]) == [1, 3, 6]
accumulateValues([1, 1, 1, 1, 1]) == [1, 2, 3, 4, 5]
accumulateValues([1, 0, 0]) == [1, 1, 1]
accumulateValues([]) == []
{% else %} 
accumulate_values([1, 2, 3]) == [1, 3, 6]
accumulate_values([1, 1, 1, 1, 1]) == [1, 2, 3, 4, 5]
accumulate_values([1, 0, 0]) == [1, 1, 1]
accumulate_values([]) == []
{% endif %}
